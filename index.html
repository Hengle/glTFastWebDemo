Notes

glTFast has a limited feature set lacking things like animation! Check the github project page for details

Built with Unity 2017.3.1f1
uncompressed with WebAssembly support.

UnityGLTF built with exceptions enabled. It would work without it, but breaks complete game in case of exceptions (like 404 errors).
